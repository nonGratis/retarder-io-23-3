|--------------------------|---|
| ***НАЗВА***:             |  Запит на пошук |
| ***УЧАСНИКИ***:          |  Користувач, система.  |
| ***ПЕРЕДУМОВИ***:        | Користувач вже знаходиться на домашній сторінці або сторінці пошуку системи. <br /> Користувач вже був авторизований. |
| ***РЕЗУЛЬТАТ***:         |  Відображення результатів пошуку користувачеві. |
| ***ВИКЛЮЧНІ СИТУАЦІЇ***: |  Невдала спроба пошуку через помилково введені дані.<br /> Користувач не був авторизований.|
| ***ОСНОВНИЙ СЦЕНАРІЙ***: |  1. Користувач вводить ключові слова або параметри пошуку. <br />   2.Користувач натискає на кнопку "Пошук".<br />  3. Система обробляє запит користувача і відображає результати, а також вносить їх в історію пошуку. <br /> 4. Користувач переглядає результати пошуку.<br />5. Користувач може взаємодіяти з результатами (наприклад, переглядати деталі, відкривати певні елементи тощо). |
