|-------------------|-------------------------------------------------|
| ***НАЗВА***:      | Реєстрація                                      |

Користувач закінчує взаємодію |

```plantuml
@startuml
|Користувач|
    start
    :Користувач починає взаємодію;
    :Користувач натискає кнопку реєстрації нового облікового запису;
|Система|
    :Система надає форму для заповнення;
|Користувач|
    :Користувач отримує форму для реєстрації і заповнює її;
    :Користувач заповнює поля: логін, адреса електронної пошти, пароль;
    :Користувач натискає кнопку підтвердження реєстрації;
        note left #FF736E

        EX.Visitor.SingUp.NotConfirm
        
        end note

|Система|
    :Система перевіряє дані з форми;
        note right #FF736E

        EX.Visitor.SingUp.Login
        EX.Visitor.SingUp.WrongEmail
        EX.Visitor.SingUp.SimplePass
        
        end note

    :Система створює новий обліковий запис;

|Користувач|
    :Користувач закінчує взаємодію;
    stop;
@enduml
```
